У Вас не выполнена настройка УКЭП на USB-носителе для работы на ЭТП МЭТС.

Вам необходимо выполнить следующие пункты:

  1. Установить программу криптопровайдер КриптоПро CSP для работы с электронной подписью
  2. Установить КриптоПро ЭЦП browser plug-in. Это программное обеспечение предназначено для создания и проверки электронной подписи на web-страницах
  3. Скачайте, откройте и добавьте в реестр настройки для Internet Explorer в REG файле. Скачать.
  4. Скачайте расширение по ссылке, затем переместите скачанный файл в любую вкладку браузера и нажмите "Установить расширение". После установки расширения обязательно закройте полностью браузер и запустите его заново.
  5. Установите расширение КриптоПро для Google Chrome. Для просмотра инструкции по установке откройте следующую ссылку https://www.cryptopro.ru/products/cades/plugin
  6. Установите расширение КриптоПро для Opera. Для просмотра инструкции по установке откройте следующую ссылку https://www.cryptopro.ru/products/cades/plugin
  7. Установите один из поддерживаемых браузеров: Mozilla Firefox, Google Chrome, Яндекс Браузер, Opera.

Для физических лиц и ИП подпись документов возможна через раздел «Госключ» в мобильном приложении «Госулуги», для этого необходимо включить соответствующий пункт в профиле. Настройка Госключа описана по ссылке https://m-ets.ru/goskey.

В мобильной версии не предусмотрена работа с Квалифицированной электронной подписью. Данный функционал доступен в полной версии, на ОС Windows, MacOS или Linux.

Требования к загружаемым изображениям лотов
Изображения должны быть хорошего качества, в высоком разрешении, минимальный размер изображений по любой из сторон должен быть не менее 500 пикселей. Рекомендуемый размер 1920x1080 и более.
Изображения должны соответствовать названию и описанию объявления, а также передавать суть вашего предложения.
Размер каждого файла не должен превышать 25 Мб.
Формат файла должен быть .jpg, .jpeg
Разрешено размещать только те изображения, на использование которых у Вас есть законное основание.
Не используйте одинаковые изображения.
Объект(ы) лота должен быть главным элементом фотографии.
Если объект предложения изображен на фоне логотипа, то такой логотип не должен занимать более 5 % фотографии.
Если на фотографии изображен человек, лицо необходимо его сделать не идентифицируемым.
После подгрузки изображений их нужно правильно сориентировать (повернуть влево или вправо)
Также, после подгрузки изображений, необходимо выбрать главное изображение, переместив его слева от первой картинки с помощью курсора.
Для удобства рекомендуется подгружать группы изображений, выделив их в открывшемся окне, после нажатия кнопки “Загрузить картинки” (не более 30 шт)
  • Запрещается использовать на подгружаемых изображениях:
  • • элементы, привлекающие внимание, например: цветные рамки, слова «срочно», «внимание» и т.п., за исключением элементов компьютерной графики, уточняющих местоположение объекта и т.п.
  • • элементы компьютерной графики, вводящие в заблуждение, искажающие реальные характеристики имущества, не несущие смысловой нагрузки
  • • логотипы, водяные знаки (вотермарки), или иные элементы других Интернет-ресурсов или брендов, названия организаций, имена и т.п.
Мы вправе автоматически наносить на загружаемые вами изображения логотипы/водяные знаки МЭТС, чтобы предотвращать несанкционированное копирование фотографий.
Мы вправе не размещать изображения по своему усмотрению, в том числе если они не соответствуют политике площадки.
